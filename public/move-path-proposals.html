Movement path preview — 30 proposals

Polyline preview like #move-path-line in the game: pointy-top odd-r hex centers with hexSize = 40, path (0,0) → (1,0) → (2,0) → (2,1) → (1,2). Game defaults from gameconfig: stroke #C77E00, width 6, draw uses GSAP stroke-dashoffset — here each row is a static or CSS-animated stand-in for comparison. 11–20 add animated flow: chevrons and dash patterns that move along the route (SVG animateMotion + CSS). 21–30 are variants of #17 (glowing chevron train): same idea, different glow, palette, density, and rhythm.

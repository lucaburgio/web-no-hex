Movement path preview — 20 proposals

Polyline preview like #move-path-line in the game: pointy-top odd-r hex centers with hexSize = 40, path (0,0) → (1,0) → (2,0) → (2,1) → (1,2). Game defaults from gameconfig: stroke #C77E00, width 6, draw uses GSAP stroke-dashoffset — here each row is a static or CSS-animated stand-in for comparison. 11–20 add animated flow: chevrons and dash patterns that move along the route (SVG animateMotion + CSS).

1 · Reference Game-style solid Same ballpark as current shipping preview: amber, 6px, ~80% opacity, round caps.
2 · Subtle Thin & muted Low visual weight for busy maps or “always on” path hints.
3 · Soft Amber glow Reads on dark terrain without much extra thickness.
4 · Medium Outlined stroke Dark underlay + bright core — extra legibility on noisy hex art.
5 · Calm pattern Static dashes Clearly “preview” rather than terrain; no motion.
6 · Directional Marching dashes Suggests flow toward the target (like a light traffic animation).
7 · Strong Gradient stroke Gold near start warming to red at the end — reads as “cost” or “commitment.”
8 · Tactical Line + arrowhead Explicit destination marker; good if players confuse last vertex with “cursor hex.”
9 · Pulsing Breathing width & glow Attention-grabbing while hovered; can feel busy if always animating.
10 · Extreme Rainbow + strobe dashes + double bloom Stress-test / joke tier — not a default for a serious map.
11 · Flow Single chevron runner One arrow glides along the polyline; rotate="auto" keeps it tangent.
12 · Flow Three chevrons, staggered Same loop with offset begin — reads like a convoy moving toward the goal.
13 · Flow Marching dashes + lead chevron CSS flow on the track plus one SMIL arrow — lane + “leader.”
14 · Flow Arrow “ticks” (CSS) Asymmetric dash pattern slides along the line — no SMIL; reads as small directional ticks.
15 · Flow Double runner (ghost + solid) Large faded chevron chases a smaller bright one — depth / urgency.
16 · Flow Five fast “droplets” Tiny chevrons, short period — busy but clearly directional.
17 · Flow Glowing chevron train Three arrows + CSS drop-shadow on fills — “energy” moving down the path.
18 · Flow Shuttle (out and back) keyPoints="0;1;0" — arrow runs to the target hex then reverses along the same trace.
19 · Flow Dual-layer motion Animated dashed spine + two staggered chevrons — line and arrows both “move.”
20 · Flow extreme Rainbow spine + four chevrons Fast dashed gradient track plus a dense chevron convoy — maximum “highway” energy.
